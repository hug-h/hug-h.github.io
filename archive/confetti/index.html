Confetti Instructions

Type into the input boxes to see your text scattered across the screen

Use commands to change the way text is displayed:

Commands must start with # and be followed by a letter and an integer

for example:

#d100 will change the density of the confetti to 100
#i30 will change the interval of the animation to 30ms (lower values are faster)
#f10 will change the font size to 10% of the viewport height
